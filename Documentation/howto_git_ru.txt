1. Сделать форк этого репозитория со своего аккаунта в гитхабе.
2. Склонировать свой репозиторий с гитхаба, а не этот, на локальную машину
    через команду git clone или через графический интерфейс.
    
/*     Предыдущие пункты нужно выполнить только единожды         */
    
3. Если репозиторий на локальной машине уже есть,
    то перед каждым началом работы нужно его обновить
    "git pull origin master", "git fetch origin master" или через гуи.
4. Создать новую ветвь проекта с названием, которое описывало бы 
    выполняемую в ветви работу.
    "git checkout -b branch_name".
5. Сделать то, что нужно было сделать. Можно делать много коммитов
    в одной ветви до ее слияния.
    Не изменять уже существующий код не касающийся данной ветви,
    не удалять файлы без необходимости:
    такие ветви сложнее объединять с проектом.
6. Добавить изменения в отслеживаемые "git add ." или через гуи и сделать
    коммит командой "git commit -m 'your commit message'" или через гуи.
7. Залить сделанную локально ветку НА СВОЙ гитхаб "git push origin branchname"
    или через гуи
8 В гитхабе в СВОЕМ репозитории нажать "New pull request", в base выбрать 
    !!!! репозиторий в Shadasviar/mario и ветвь master !!!!, со своей стороны
    выбрать свою ветвь для коммита.
9. Ждать применения запроса и слияния/отказа в слиянии ветви с проектом.
